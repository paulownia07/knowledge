# Install

## Windows

1. [公式サイト](https://www.postgresql.org/)へ移動
2. downloadからOS選択
3. インストールディレクトリの選択
4. データを保存するディレクトリの選択
5. 管理者ユーザのパスワードを設定する
6. ポート番号の設定
7. ロケールを設定する
8. 設定ファイル(postgresql.conf)を書き換える

---

## Linux (Debian系)

1. 以下のコマンドを実行する
   ```
   sudo apt install -y postgresql postgresql-contrib
   ```
