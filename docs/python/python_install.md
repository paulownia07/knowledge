# Install

## Windows

1. [公式サイト](https://www.python.org/)からダウンロード
2. パスを自動で通すようにしてインストール
3. インストールに成功したか確認する
   ```
   python --version
   ```

---

## Ubuntu

1. python x.yをインストールする
   ```
   sudo yum install pythonxy
   ```
2. インストールに成功したか確認する
   ```
   python --version
   ```
