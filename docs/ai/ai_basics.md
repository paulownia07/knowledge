### AI モデル

| model  | developer     | description                                          |
| ------ | ------------- | ---------------------------------------------------- |
| GPT    | OpenAI        | 文章生成、コーディング、推論、エージェント機能が得意 |
| Claude | Anthropic     | 長文処理とコード生成が得意                           |
| Gemini | Google        | テキスト・画像・音声・動画を統合的に扱うことが可能   |

| Qwen   | Alibaba Cloud | 多言語対応が強く、中国語性能が特に高い               |
