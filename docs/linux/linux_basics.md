# basics

## ディストリビューション

主なLinuxのディストリビューションは「Red Hat系」、「Debian系」の2種類存在する。

### Red Hat系

- Red Hat Enterprise Linux
- Cent OS / Cent OS Stream
- Fedora
- Alma Linux

### Debian系

- Debian GNU/Linux
- Ubuntu

> [!NOTE]
> Amazon LinuxはRed Hat系/Debian系のどちらか明らかにされていないらしい。

---

## ユーザ

Linuxにおけるユーザは「rootユーザ」、「一般ユーザ」に分けられる。

- rootユーザ : システム管理が可能なユーザ。
- 一般ユーザ : 操作に制限があるユーザ。

---

## 基本的なディレクトリ構造

Linuxのファイルシステム内のレイアウトはFHS(Filesystem Hierarchy Standard)として標準化が進められており、主要なディストリビューションはFHSをサポートしている

| path     | description                                                                    |
| -------- | ------------------------------------------------------------------------------ |
| `/etc`   | システムやアプリケーションの設定情報が配置されるディレクトリ                   |
| `/home`  | 一般ユーザのホームディレクトリ                                                 |
| `/root`  | ルートユーザのホームディレクトリ                                               |
| `/boot`  | 起動に必要な設定やカーネルイメージが配置されるディレクトリ                     |
| `/bin`   | 一般ユーザが実行可能なコマンドが配置されるディレクトリ                         |
| `/sbin`  | ルートユーザのみが実行可能なコマンドが配置されるディレクトリ                   |
| `/lib`   | 共有ライブラリやカーネルモジュールが配置されるディレクトリ                     |
| `/dev`   | デバイスファイルが配置されるディレクトリ                                       |
| `/usr`   | コマンドやユーティリティなどが配置されるディレクトリ                           |
| `/var`   | ログファイルなどの頻繁に書き換えられるファイルが配置されるディレクトリ         |
| `/tmp`   | 一時的なファイルが配置されるディレクトリ                                       |
| `/mnt`   | 一時的にマウントするファイルシステムのマウントポイントが配置されるディレクトリ |
| `/media` | リムーバルメディアのマウントポイントが配置されるディレクトリ                   |
| `/opt`   | パッケージ管理の仕組みを使ってプログラムがインストールされるディレクトリ       |
| `/proc`  | カーネル内部の情報にアクセスするための仮想的なファイルが配置されるディレクトリ |

## 基本的なパス

| path          | description                                                                      |
| ------------- | -------------------------------------------------------------------------------- |
| `/etc/passwd` | ユーザアカウント情報                                                             |
| `/etc/shadow` | パスワード情報                                                                   |
| `/etc/group`  | グループの設定                                                                   |
| `/var/chache` | 一時的なキャッシュファイルが配置されるディレクトリ                               |
| `/var/lock`   | アプリケーションが排他制御に使用するロックファイルが配置されるディレクトリ       |
| `/var/log`    | ログファイルが配置されるディレクトリ                                             |
| `/var/run`    | システムの状態を示すファイルが配置されるディレクトリ                             |
| `/var/spool`  | 処理待ちのデータが配置されるディレクトリ                                         |
| `/usr/bin`    | ユーザが一般的に使用するコマンドが配置されるディレクトリ                         |
| `/usr/sbin`   | システム管理者が使用するコマンドが配置されるディレクトリ                         |
| `/usr/lib`    | プログラムに必要な共有ライブラリが配置されるディレクトリ                         |
| `/usr/local`  | ローカルシステムのコマンドやライブラリやドキュメントなどが配置されるディレクトリ |
| `/usr/share`  | システムアーキテクチャに依存しないファイルが配置されるディレクトリ               |
| `/usr/src`    | カーネルソースなどのソースファイルが配置されるディレクトリ                       |

---

## デバイス情報を格納しているパス

| path              | description      |
| ----------------- | ---------------- |
| `/proc/cpuinfo`   | CPU情報          |
| `/proc/interupts` | IRQ情報          |
| `/proc/ioports`   | I/Oアドレス情報  |
| `/proc/meminfo`   | メモリ情報       |
| `/proc/swaps`     | スワップ領域情報 |
| `/proc/bus/usb/*` | USBデバイス情報  |
| `/proc/bus/pci/*` | PCIデバイス情報  |

---

## ネットワーク情報を格納しているパス

| path                 | description                                          |
| -------------------- | ---------------------------------------------------- |
| `/etc/hostname`      | ホスト名が記述されているファイル                     |
| `/etc/hosts`         | ホスト名とIPアドレスとの対応が記述されているファイル |
| `/etc/resolv.conf`   | 参照先DNSサーバが記述されているファイル              |
| `/etc/nsswitch.conf` | 名前解決の順序が記述されているファイル               |

---

## アカウント情報を格納しているパス

| path                   | description                                                          |
| ---------------------- | -------------------------------------------------------------------- |
| `/etc/passwd`          | ユーザアカウント情報が記述されているファイル                         |
| `/etc/shadow`          | パスワード情報が記述されているファイル(ルートユーザのみ読み取り可能) |
| `/etc/group`           | グループ情報が記述されているファイル                                 |
| `/etc/default/useradd` | useraddコマンドのデフォルト値が保存されているファイル                |
| `/etc/skel`            | ホームディレクトリ作成時に配布するファイルを配置するディレクトリ     |
| `/etc/nologin`         | ルートユーザ以外によるログイン以外を禁止するためのファイル           |
